半灯守り

は ん と う も り  —  丸 め な い 夜 の ゲ ー ム
世界が溶けて、灯は「点く / 消える」をやめた。いまはどれも、0 と 1 のあいだの小数で点っている。

あなたは 半灯守り。夜ごとに灯を見て回り、落ちすぎたものに油を足す。役所は 丸め令 を出した——〇.二未満は零とみなして消してよい、と。だが、丸めるかどうかはあなたが決める。

零に丸められず静かに落ちた灯は、腐葉土になる。土は温く、次の灯を育てる。丸めた灯は、土にならない。

勝ち負けはない。あるのは、世界がどれだけ 帯（Float） のままでいられたか、だけ。
