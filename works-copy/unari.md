# 〰️ うなり / UNARI

## なにこれ一言
音程を「半音きざみ（離散）」から「微分音（連続）」まで一本のつまみで行き来し、二つの音のズレを“うなり”として耳で聴くツール。音版の Boolean→Float。

## 特徴3つ
- 量子化↔連続を補間: つまみを回すと、ピアノの半音グリッド（離散）と、グリッドを無視した微分音（連続）の間を cents 空間でなめらかに補間。音律の「あいだ」に触れる。
- ズレが“うなり”になる: 二音の cents 差が、うなり（beat Hz）として可聴化される。理屈でなく、揺れる音そのものでズレを体感できる。
- Web Audio・依存ゼロ: ライブラリなしの素の Web Audio。三重がけ命名「うなり」（唸り＝音／うなり＝beat／たわむ）。

## つかいかた3手順
1. 二つの音を鳴らす
2. 量子化↔連続つまみを回して、半音と微分音のあいだを探す
3. 二音の cents 差が生む“うなり”の速さを聴き分ける

## FAQ
- Q 音楽の知識が要る？ A 不要。回して聴くだけで「ズレ＝うなり」が体でわかる。
- Q 何かインストールする？ A しない。ブラウザの Web Audio だけで鳴る。
